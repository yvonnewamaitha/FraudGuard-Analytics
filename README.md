# FraudGuard Analytics

FraudGuard Analytics is a fintech data analytics platform designed to detect fraudulent online payment transactions using a Decision Tree machine learning model.

## Features
- Fraud prediction using Decision Tree algorithm
- Transaction risk analysis
- Visual analytics dashboard
- Explainable AI decision rules

## Technology Stack
- Python
- Scikit-learn
- Pandas
- Google Colab
- Streamlit (prototype)

## Academic Purpose
This project was developed as part of a Data Analytics portfolio demonstrating how machine learning can solve real-world financial fraud detection problems.

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Input transaction data to test fraud detection
